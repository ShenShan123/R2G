# Instruction for Routing Homograph Merging

## Applicable Scripts

1. **node_merge_homographs.py**
   - Merges node features from multiple routing homographs
   - Concatenates all node features with graph_id assignment
   - Input: Individual homograph files (e.g., design1_B_homograph.pt, design2_B_homograph.pt)
   - Output: Merged homograph with all node features

2. **edge_merge_homographs.py**
   - Merges edge features and labels from multiple routing homographs
   - Concatenates all edge features with updated edge indices
   - Input: Individual homograph files
   - Output: Merged homograph with all edge features and labels

## Usage

1. Run node_merge_homographs.py first to merge node features
2. Run edge_merge_homographs.py second to merge edge features
3. Output file: route_X_homograph.pt (where X is the view: B, C, D, E, F)

## Graph Type

This directory handles **homographs** (graphs with unified node/edge types) for routing task.
