# Instruction for Placement Heterograph Merging

## Applicable Scripts

1. **node_merge_heterographs.py**
   - Merges node features from multiple placement heterographs
   - Concatenates all node features with graph_id assignment
   - Maintains node type information (gate, io_pin, net, pin)
   - Input: Individual heterograph files (e.g., design1_B_heterograph.pt, design2_B_heterograph.pt)
   - Output: Merged heterograph with all node features

2. **edge_merge_heterographs.py**
   - Merges edge features and labels from multiple placement heterographs
   - Concatenates all edge features with updated edge indices
   - Maintains edge type information (gate_gate, gate_net, etc.)
   - Input: Individual heterograph files
   - Output: Merged heterograph with all edge features and labels

## Usage

1. Run node_merge_heterographs.py first to merge node features
2. Run edge_merge_heterographs.py second to merge edge features
3. Output file: place_X_heterograph.pt (where X is the view: B, C, D, E, F)

## Graph Type

This directory handles **heterographs** (graphs with typed nodes and edges) for placement task.
